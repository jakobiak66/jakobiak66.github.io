# Kundenbilder – Haustechnik Hun

Legen Sie hier die Kundenbilder ab. Verwenden Sie die gleichen Dateinamen wie in image_map.json.
Diese Bilder ersetzen automatisch die heruntergeladenen Originalbilder.

## Benötigte Bilder

| Dateiname | Beschreibung |
|-----------|--------------|
| `logo.png` | Firmenlogo (PNG mit Transparenz bevorzugt) |
| `hero.jpg` | Hauptbild für die Startseite (mind. 1920x800px) |
| `team-laszlo.jpg` | Foto von Laszlo Hun |
| `pelletsheizung.jpg` | Bild Holz-/Pelletsheizung |
| `leistungen.jpg` | Bild für Leistungsübersicht |
| `fahrzeug.jpg` | Fahrzeug oder Baustelle |

## Hinweis
Alle Bilder werden in `/website/public/images/` eingebunden.
